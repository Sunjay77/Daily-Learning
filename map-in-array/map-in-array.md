## Map in array

- map () method is a common feature in languages like javascript and php. It creates a new array by applying a function to every element of an existing array.

## key Characyeristic 

- Non-mutating: It does not change the original array; it returns a new one
- Transformation: Useful for modifying data, such as doubling numbers or extracting properties from objects.
- vs. forEach(): Use map() when you need the resulting array; use forEach() for side effects like logging.

## Array of Map Data Structures

- This refers to a collection where every element in the array is itself a Map (a key-value pair collection)

- C++ Example: In C++, you can declare an array of maps to store complex associated data in a list format.

- JavaScript Map Object: A Map is an object that holds key-value pairs where keys can be of any type. You can store multiple such objects in a standard array.

- Conversion: You can convert a Map object into an array using the spread operator ([...myMap]) or Array.from(myMap).
